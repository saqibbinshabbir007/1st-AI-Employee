- `skill: customer-inquiry` likho

### 2. Naya Vehicle List Karna Ho:
- `Inbox/` mein file banao

### 3. Daily Report Chahiye:
- `Inbox/` mein file banao

- Save karo — Atlas report banayega

- Content paste karo
- Save karo — Atlas summary banayega
