# Skill: sales-report
**Version:** 1.0
**Business:** Car Markaz
**Last Updated:** 2026-03-26

---

## Purpose
Car Markaz ke liye daily ya weekly sales report generate karna. Muhammad Saqib raw data drop kare, AI formatted business report banaye.

## Trigger Conditions
Task file mein yeh keywords hon: `report`, `sales`, `daily`, `weekly`, `farokht`, `summary`, `performance`

---

## Step-by-Step Instructions

**Step 1: Report Period Identify Karo**
Task file se determine karo:
- Daily Report (ek din ka) ya Weekly Report (7 din ka)?
- Date range: [start date] se [end date] tak

**Step 2: Input Data Parse Karo**
Task file se yeh data extract karo:

```
SALES:
- Vehicles sold: [count]
- Vehicle details: [model, price each]
- Total revenue: PKR [amount]

INQUIRIES:
- New inquiries received: [count]
- Source breakdown: WhatsApp [x] / Phone [y] / Walk-in [z] / Online [w]

ACTIVITIES:
- Follow-ups completed: [count]
- Test drives conducted: [count]
- Showroom visits: [count]

PIPELINE:
- Pending deals (almost finalized): [count + brief]
- Active leads: [count]
- Cold leads: [count]
```

**Step 3: Metrics Calculate Karo**
- Total Revenue = sum of all sold vehicles
- Average Selling Price = Total Revenue / Vehicles Sold
- Inquiry-to-Visit Rate = (Showroom Visits / Total Inquiries) × 100
- Visit-to-Sale Rate = (Vehicles Sold / Showroom Visits) × 100
- Most inquired vehicle type

**Step 4: Previous Report Compare Karo (Agar Available Ho)**
- `Done/` folder mein previous period report dhundo
- Agar mila → compare karo:
  - Revenue change: +/- PKR [amount] ([percentage]%)
  - Inquiry change: +/- [count]
  - 30% se zyada change → ALERT flag karo

**Step 5: Recommendations Generate Karo**
Data ke basis pe suggestions:
- Agar kisi vehicle mein zyada interest → "Stock add karne ki zaroorat"
- Agar inquiries kam → "Marketing boost ki zaroorat"
- Agar pending deals zyada → "Follow-up focus karna chahiye"

**Step 6: Report Banao**

Full Report format:

**Step 7: Output Save Karo**
- Full report: `Plans/Reports/REPORT_[period]_[date].md`
- Executive summary only: `Pending_Approval/REPORT_SUMMARY_[date].md`

**Step 8: Cleanup**
- Original task file `Done/` mein move karo
- Dashboard.md update karo
- Log entry karo

---

## Output Destinations
- Full report: `Plans/Reports/REPORT_[DAILY/WEEKLY]_YYYY-MM-DD.md`
- Summary for owner: `Pending_Approval/REPORT_SUMMARY_YYYY-MM-DD.md`

## Escalation Conditions
- Revenue 50% se zyada down → Needs_Action URGENT
- Koi fraud ya unusual transaction → Needs_Action URGENT
- Data incomplete ya inconsistent → Needs_Action for clarification
